# Self-Employed Work Management Platform

A comprehensive work management platform for self-employed professionals with React frontend, Spring Boot microservices backend, Python AI agent, and full Docker containerization.

## Project Structure

```
MyWorkManagement/
├── frontend/           # React TypeScript frontend application
├── backend/           # Spring Boot microservices
│   ├── company-service/
│   ├── project-service/
│   ├── task-service/
│   └── analytics-service/
├── ai-agent/          # Python FastAPI AI agent
├── docker/            # Docker configuration files
└── docs/             # Project documentation
```

## Tech Stack

### Frontend
- React 18
- TypeScript
- Tailwind CSS
- React Router
- Axios
- Chart.js/Recharts

### Backend
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Spring Cloud OpenFeign
- Spring Validation

### AI Agent
- Python 3.11
- FastAPI
- OpenAI API
- SQLAlchemy
- Pandas
- NumPy

## Getting Started

### Prerequisites
- Node.js 18+
- Java 17+
- Python 3.11+
- Docker and Docker Compose
- PostgreSQL
